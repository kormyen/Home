Megamex is a lens for overviewing megaliths

2018-09-30 - 2018
21 hours since 2018 Sep

I started a project of cataloging all the megalithic sites hoping to add tags of similar construction techniques, themes and details but I very quickly realised that there are many many many more amazing ancient megalithic sites than I imagined.

Did you know there are 44 large pyramids in Egypt for starters?

I plan to come back to this at some point - visualize it something like Memex, make it open source and make it easy for people to add new sites and media.

I cannot finish this by myself.
